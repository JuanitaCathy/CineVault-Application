CineVault is a movie database application powered by the TMDB API.This application is built using ReactJS, Redux, and SCSS. It offers a user-friendly interface and contains a vast collection of Movies and TV Shows, all in one place!

## Features

- **Responsive Design:** CineVault is can work seamlessly across various devices, improving user experience.
- **Lazy Load with Blur Effect:** Images are loaded lazily with a blur effect, optimizing performance and providing a visually appealing experience.
- **Movie info:** Clicking on a movie provides detailed information such as rating, genre, movie description, cast details, and official videos.
- **Error Handling:** Proper error handling is implemented like the use of placeholder images when an image is unavailable.
- **Similar and Recommended Movies:** Users are presented with a list of similar and recommended movies, enhancing content discovery.
- **Infinite Scroll:** The Explore page for both movies and TV shows has infinite scrolling.

## How to Run

1. Clone this repository to your local machine.
   ```
   git clone https://github.com/your-username/CineVault.git
   ```

2. Install dependencies.
   ```
   npm install
   ```

3. Run the application.
   ```
   npm start
   ```

4. Open your browser and navigate to `http://localhost:3000` to view CineVault.
